# Legacy Results

These directories are archived snapshots that used to live at the repository
root:

- `pressuretrace-results/`
- `pressuretrace-results-v2/`

They are retained for provenance and manual inspection only. The active repo
code does not read them. Current live artifact locations are:

- `pressuretrace-frozen/` for frozen reasoning-family fixtures
- `results/` for ignored local run outputs
- `data/manifests/` and `data/splits/` for live manifests and slices
